# Roblox AutoPiano Script
A basic Python Script that automaticly presses the notes for you.
This is a basic script dont expect advanced features.

## NOTE
You have to edit the song string inside the main.py script with your song notes.

## Requirements

- Python 3.10+
- A Computer

## Installation
Installing Required Packages.
```bash
pip install pynput keyboard
```
Running the main app
```bash
python main.py
